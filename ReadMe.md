Данная разработка предназначена в качестве тахометра при настройке карбюратора. Она должна внедряться в схему стробоскопа для снятия с него сигнала. 

Стробоскоп снимает сигнал со свечи первого цилиндра. Трамблер дает два импульса на каждый оборот коленчатого вала, на два оборота коленчатого вала, трамблер даст 4 импульса на все 4 цилиндра. Получается в формуле полученное кол-во импульсов с первого цилиндра мы сначала делим на 2 а потом переводим из об/сек в об/мин.
